## Purpose

Internal dashboard to monitor KPIs of a SaaS platform in real time: active users, hourly revenue, 5xx errors, p95 latency.

## Technical decisions

- **Go + WebSockets** for the stream: 5-second aggregations instead of pushing every raw event to the client.
- **D3.js** for custom charts; we dropped Recharts because we needed smooth transitions between time windows.
- **Backpressure on the client**: when the tab is in the background, the server reduces frequency to 1/min.

## Challenges

Avoiding memory leaks when switching time windows (1h, 24h, 7d). Implemented a ring buffer per series with explicit dispose of the Resize observers.
